## ✅ WL Grand Cherokee 4xe — Buyer Inspection Checklist

### 🔋 4xe Plug-In Hybrid System (TOP PRIORITY)
- [ ] **Charge port door opens/closes smoothly** (they sometimes stick)
- [ ] **Vehicle charges without interruption** on both Level 1 & Level 2
- [ ] **No “Service Hybrid System” messages** in cluster
- [ ] EV mode engages normally at startup
- [ ] Hybrid → gas transition is **smooth and quiet**
- [ ] Electric range estimate is ~20–25 miles when fully charged
- [ ] Regenerative braking feels smooth (not grabby)
- [ ] Battery cooling fan is not excessively loud after driving
- [ ] Check under rear seat/cargo area for **water intrusion near battery compartment**

---

### ⚙️ Known 4xe Mechanical Watch-Items
- [ ] No hesitation when accelerating from stop (early software issue)
- [ ] Transmission shifts smoothly (ZF 8-speed should feel very refined)
- [ ] No clunking when switching between EV and hybrid drive
- [ ] Engine start during hybrid transition is **not harsh or delayed**

---

### 💻 Uconnect 5 Infotainment (Common Problem Area)
- [ ] System boots within ~10 seconds
- [ ] No screen lag, ghost touches, or freezing
- [ ] Wireless CarPlay/Android Auto connects quickly
- [ ] 360° camera loads instantly (not black screen)
- [ ] Check for recent software updates installed

---

### 🛞 Quadra-Lift Air Suspension (If Equipped)
- [ ] Vehicle raises/lowers smoothly in all modes
- [ ] No “Air Suspension Service Required” messages
- [ ] Listen for loud compressor noise
- [ ] Vehicle sits level after being parked overnight
- [ ] No sagging at any corner

---

### 🛡️ Driver Assist & Tech Features
- [ ] Adaptive cruise works smoothly (no sudden braking)
- [ ] Lane centering holds position correctly
- [ ] Night vision (if equipped) displays clearly
- [ ] Park assist functions normally

---

### 🚙 Interior-Specific Checks
- [ ] Massaging seats operate correctly
- [ ] Seat ventilation actually blows cold air
- [ ] McIntosh audio (if equipped) has no speaker distortion
- [ ] Ambient lighting zones all function
- [ ] Panoramic roof shade doesn’t bind or squeak

---

### 🧊 Climate System (Known Quirk Area)
- [ ] Heat works quickly in EV mode
- [ ] No clicking sounds from blend doors
- [ ] Rear climate controls respond correctly

---

### 🛠️ Recalls / Software Updates to Confirm
- [ ] Hybrid battery fire-risk recall completed
- [ ] Hybrid control module updates installed
- [ ] Transmission software updates applied

---

### 🛞 Exterior & Chassis
- [ ] Undercarriage shows no battery shield damage
- [ ] Orange high-voltage cables intact and secure
- [ ] Tow hooks and skid plates not bent
- [ ] Tires wear evenly (heavy vehicle → sensitive to alignment)
- [ ] Tow hitch isn't overly rusty (if equipped)
- [ ] Trailer wiring is operational (if equipped)

---

### 🚩 Major Red Flags
- Hybrid system warning lights
- Rough EV → engine transitions
- Air suspension sagging overnight
- Frequent infotainment crashes
- Battery unable to fully charge

---

### 💰 Value-Protection Checks
- [ ] Confirm **8yr / 100k mile hybrid battery warranty** still active
- [ ] Verify original Level-1 charging cable included
- [ ] Ask if vehicle was frequently fast-charged vs home-charged

---

## 🧠 Quick Buyer Reality Check
A healthy 4xe should feel:
- Extremely smooth
- Very quiet in EV mode
- Seamless when switching power sources

If it feels rough, noisy, or glitchy — something is wrong.
